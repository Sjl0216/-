# TabBar 图标说明

本目录用于存放底部导航栏的图标文件。

## 所需图标文件

请准备以下 8 个图标文件（建议尺寸：81x81 像素）：

### 首页
- `home.png` - 首页图标（未选中）
- `home-active.png` - 首页图标（选中状态，紫色）

### 记录
- `record.png` - 记录图标（未选中）
- `record-active.png` - 记录图标（选中状态，紫色）

### 科普
- `encyclopedia.png` - 科普图标（未选中）
- `encyclopedia-active.png` - 科普图标（选中状态，紫色）

### 设置
- `settings.png` - 设置图标（未选中）
- `settings-active.png` - 设置图标（选中状态，紫色）

## 图标规范

- **尺寸**：建议 81x81 像素（最大 40kb）
- **格式**：PNG 格式，支持透明背景
- **颜色**：
  - 未选中：灰色 (#999999)
  - 选中：紫色 (#667eea)

## 临时解决方案

如果暂时没有图标，可以：
1. 使用在线图标生成工具（如 iconfont.cn）
2. 或者暂时使用 logo.png 代替
3. 后续替换为正式图标

## 使用 iconfont 制作图标步骤

1. 访问 https://www.iconfont.cn/
2. 搜索图标：home、list、book、setting
3. 下载 PNG 格式
4. 修改颜色并保存为对应文件名
5. 放入此目录
